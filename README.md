# Code Studio

Code studio web dasturi yordamida siz dasturlash tillarida kod yozishib o'rganishingiz mumkin.

Ushbu dastur dasturlashni endi o'rganayotgan insonlar uchun va interview topshirayotgan isonlar uchun juda ham samarali.

## Code Studio Community

Code yozib o'rganishdan tashqari, siz yozgan kodlaringizni yaqin do'stlaringizga ham ulashish imkoniyati mavjud.

## Programming Languages

Endilikda siz Javascript, Typescript, Java, C++, Python, C#, Ruby, Go dasturlash tillarida kod yoza olasiz.
